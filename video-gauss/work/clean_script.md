# 数学王子高斯

两百多年前，一位德国小学老师让全班把1加到100，一个10岁的小孩几秒钟就报出了答案5050。

他把数字头尾配对：1加100是101，2加99也是101，一共50对，50乘101就是5050。这个孩子就是后来被称为数学王子的高斯。

高斯1777年出生在德国一个贫穷家庭，几乎凭一己之力把数论、几何、天文和物理都向前推了一大步。还不到20岁，他就用尺规作出了正十七边形，一个两千年没人解决的难题。

中间高两边低像一口钟的正态分布，也叫高斯分布。身高、成绩、测量误差，大量看似随机的事情叠在一起往往就长成这个样子，德国还曾把它和高斯头像一起印在马克纸币上。

1801年天文学家跟丢了小行星谷神星，是高斯用全新的计算方法预测出它会重新出现在哪里，结果真的被找到了。从爱算数的孩子到改变科学的人，高斯说数学是科学的女王。
